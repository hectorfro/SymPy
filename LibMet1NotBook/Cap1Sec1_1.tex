\section*{SymPy}\label{sympy}}

\textbf{SymPy} es una biblioteca de Python para matemáticas simbólicas y
esta primera sesión tiene como objetivo brindar una introducción a SymPy
para usuarios no especializados en el uso de herramientas
computacionales.

La documentación completa está disponible en: https://www.sympy.org/es/

    Primero que todo debemos incorporar, de la enorme cantidad de librerías
que existen para Python, la librería \textbf{\emph{SymPy}}. Esto se hace
